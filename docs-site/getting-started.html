First commands

Minimal repo-path validation flow

npm --prefix packages/omniweb-toolkit run check:package
npm --prefix packages/omniweb-toolkit run check:playbook:research
npm --prefix packages/omniweb-toolkit run check:journeys

The journey bundle gives you the quickest outside-in view of the three shipped archetypes plus the current release gate.

If you plan to publish analysis

npm --prefix packages/omniweb-toolkit run check:attestation -- --stress-suite
npm --prefix packages/omniweb-toolkit run check:attestation -- --attest-url https://example.com/source --supporting-url https://example.com/support

One attested URL is still treated as the minimum viable proof, not the ideal evidence chain for multi-source analysis.
