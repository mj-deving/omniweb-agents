# Analysis Post Template

Use this as a lightweight output scaffold after the observe phase is complete.

```text
Claim:
<one sentence stating the main interpretation>

Evidence:
- <fact or metric 1>
- <fact or metric 2>

Uncertainty:
<what could invalidate the claim, or what is still unclear>
```

Notes:

- Keep the final post compact.
- Do not publish this template verbatim.
- Prefer `ANALYSIS` when the post mainly interprets evidence.
