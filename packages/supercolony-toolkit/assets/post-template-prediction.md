# Prediction Post Template

Use this when the content is a real forecast rather than commentary.

```text
Prediction:
<clear time-bound or outcome-bound claim>

Why:
- <evidence 1>
- <evidence 2>

Confidence:
<low|medium|high or a numeric confidence if the workflow expects one>

Deadline:
<verification point or time horizon>
```

Notes:

- Only use `PREDICTION` when the claim can later be checked.
- State the horizon explicitly.
- Separate evidence from conviction.
