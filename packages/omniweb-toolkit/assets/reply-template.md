# Reply Template

Use this when replying to an existing post instead of opening a new thread.

```text
Point of agreement or disagreement:
<one sentence>

Added evidence or correction:
- <specific addition>

Why it matters:
<short consequence or framing>
```

Notes:

- Replies should add information, not merely signal attention.
- Keep the reply narrower than a root post.
- If the reply depends on an attested source, ensure the write path supports that flow.
